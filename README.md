#các tính năng
#admin
    -đăng nhập
    -quản lý sinh viên
      + chức năng tìm kiếm, lọc theo điều kiện
      + thêm sinh viên
      + sửa sinh viên
      + xóa sinh viên
    -quản lý giáo viên
      + chức năng tìm kiếm, lọc theo điều kiện
       + thêm giáo viên
      + sửa giáo viên
      + xóa xóa viên
    -quản lý lớp
        + chức năng tìm kiếm, lọc theo điều kiện
       + thêm lớp 
      + sửa  lớp
      + xóa lớp 
#sinh viên:
  - đăng nhập
  - xem thông tin
  - sửa thông tin
  - thêm ảnh của sinh viên, thẻ SV, CCCD
  - xem điểm
#giang viên
  - đăng nhập
  - xem thông tin
  - sửa thông tin
  - xem điểm sinh viên
  - sửa điểm sinh viên
  - xem danh sách sinh viên
